# منصة عشة التعليمية

موقع تعليمي حديث ومتجاوب لعرض الدروس والمحتوى التعليمي.

## المميزات

- 🎨 تصميم عصري وجذاب
- 📱 متجاوب مع جميع الأجهزة
- 🌙 واجهة سهلة الاستخدام
- 📚 عرض الدروس مع الصور والإنفوجرافيك
- 🎥 مشغل فيديو مدمج
- 📝 اختبارات تفاعلية
- 🔗 روابط لأدوات تعليمية خارجية
- ⚡ أداء سريع وتحميل محسّن

## المحتوى

### الدروس
- **الدرس الأول**: مقدمة في تطوير الويب
- **الدرس الثاني**: قواعد البيانات

### الموارد
- صور توضيحية لكل درس
- إنفوجرافيك تعليمي
- فيديوهات شرح
- ملفات PDF و DOCX قابلة للتحميل

### الروابط المفيدة
- Gemini AI - مساعد ذكي
- Animaker - إنشاء الرسوم المتحركة
- Kahoot - اختبارات تفاعلية
- Genially - محتوى تفاعلي

## النشر على Vercel

### الطريقة الأولى: من خلال GitHub

1. ارفع المشروع إلى GitHub
2. اذهب إلى [Vercel](https://vercel.com)
3. سجل دخول أو أنشئ حساب جديد
4. اضغط على "New Project"
5. اختر المستودع من GitHub
6. اضغط على "Deploy"

### الطريقة الثانية: من خلال Vercel CLI

```bash
# تثبيت Vercel CLI
npm i -g vercel

# تسجيل الدخول
vercel login

# نشر المشروع
vercel
```

### الطريقة الثالثة: السحب والإفلات

1. اذهب إلى [Vercel](https://vercel.com)
2. اسحب مجلد المشروع وأفلته في صفحة Vercel
